# Classbook Frontend

## 環境構築
### Requirements

- Node.js (See [`.node-version`](#) file)

### パッケージのインストール

```
node -v // v22.11.0
npm i --no-save
```

## ローカル環境確認用スクリプト

### ブラウザ確認

```
npm run dev
```

ブラウザで http://localhost:5173/ を開いてください。

### テスト確認

* jestでユニットテスト書く想定

```
npm run test
```
